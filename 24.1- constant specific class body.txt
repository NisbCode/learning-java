Para que apenas uma dessas variáveis sejam de um tipo diferente
Pelo visto é raro de ser usado, maaaassss, tá aí

A q será mudada será a ultima colocada DESCOBRI TENTANDO, CONFERE UM DIA AÍ PF


Isso delimita que todas as variáveis constantes serão do tipo A

    public String getId() {
        return "A";
    }

Porém, caso vc queira que uma delas seja B, ao fim das variáveis constantes coloque:

    PESSOA_FISICA(1, "Pessoa Física"), PESSOA_JURIDICA(2, "Pessoa Jurídica") {
        public String getId() {
            return "B";
        }
    };


A ultima colocada, no caso a Jurídica, será a B
